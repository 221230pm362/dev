요소선택 (추가선택자)

        선택자[속성]     : 해당 속성 가진 모든 요소 선택 
        선택자[속성=값]  : 속성 === 값 선택 
        선택자[속성~=값] : 값 에 해당하는 속성 있으면 선택 
        선택자[속성^=값] : 시작이 값 
        선택자[속성$=값] : 끝 
        선택자[속성*=값] : 포함 

        :타입명 // type이 "타입명" 에 해당하는 요소를 가져옴
        :text
        :button
        :checkbox
        :file
        :reset
        :password

        input:checked
        option:selected

        button:enabled
        button:disabled
