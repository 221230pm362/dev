4) 화살표 함수

        function 키워드 대신 => 를 사용

        [문법]
            - 매개 변수가 없을 때 : () => { ... }
            - 매개 변수가 하나 일 때 : x => { ... }
            - 매개 변수가 두 개 이상일 때 : (x, y) => { ... }
            - 리턴 처리
                - 처리할 라인이 여러 개라면 {}로 감싸서 처리해야 한다.
                (x, y) => {
                    실행 구문;

                    return x * y;
                }
                - 처리할 라인이 하나라면 {}와 return 문을 생략할 수 있다.
                (x, y) => x * y;
        
        * 기존 function과 this가 가리키는 대상이 조금 다르기 때문에 주의해야 한다. 
        (javascript arrow function this)
