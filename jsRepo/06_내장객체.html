내장객체

Object

        자바스크립트의 가장 기본적인 내장 객체
        Object라는 생성자 함수를 통해서 만들어진 인스턴스이다.

Number

        정수 실수 저장하는 객체

        toExponential
        toPrecision
        toFixed

String

        문자열을 저장하는 객체
        문자열 관련 메소드들을 제공

        str.bold();
        str.italics();
        str.fontcolor('red');
        str.link('https://www.naver.com');

Date

        날짜와 시간 정보를 저장하는 객체
